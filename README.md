# machine-learning-projekt
Klasyfikacja wieku na podstawie wyników testów poznawczych (LEMON dataset)

# Opis
Projekt polega na analizie danych psychometrycznych oraz zbudowaniu modelu klasyfikacyjnego, który przewiduje przynależność osoby do grupy wiekowej (młodzi vs starsi) na podstawie wyników testów poznawczych.

# Cel
Udowodnienie, że różne aspekty funkcjonowania poznawczego (inteligencja płynna vs skrystalizowana vs funkcje wykonawcze) pozwalają odróżnić grupy wiekowe.

# Technologie
- Python (pandas, sklearn)
- Jupyter Notebook (Google Colab)

# Jak uruchomić
1. Biblioteki (pip install pandas scikit-learn jupyter)
2. Pliki (Participants_LEMON.csv, LPS.csv, WST.csv, TMT.csv) muszą się znajdować w tym samym folderze
3. Komórki powinny się wykonywać

# Czego się nauczyłem
- integracji danych z wielu plików
- czyszczenia, przygotowywania oraz standaryzacji danych
- budowy prostego modelu klasyfikacyjnego (Logistic Regression)
- podziału danych na zbiór treningowy i testowy
- ewaluacji użytego modelu
